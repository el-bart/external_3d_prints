# cylinder lock

from [printables](https://www.printables.com/model/42686-cylinder-lock-fully-functional-parametric).

recommended print settings from the author:
* 0.2mm 1st layer
* 0.15mm other layers
* 4 springs 4x19mm (or longer)

`refined_key.scad` is a version of the default key, that better matches the model (with extra clearances).
